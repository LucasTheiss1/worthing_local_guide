# Worthing Local Guide

A small static website built with HTML and CSS to help visitors and new residents discover Worthing quickly.

## User Value
This website helps users:
- Find quick ideas for places to visit
- Plan a simple weekend itinerary
- Access information with clear navigation across pages

## Features
- 3-page site: Home, Places, Contact
- Responsive layout (mobile/tablet/desktop)
- Clean navigation and consistent header/footer
- Contact form (front-end only)

## Technologies Used
- HTML5
- CSS3
- Copilot
- VsCode
- Balsamiq

## Testing
- Checked that all navigation links work between pages
- Verified responsive layout using browser dev tools (mobile + tablet widths)
- Validated HTML using a HTML validator (optional)
- Verified contrast and readable font sizing
